# 競馬予想 評価ルール（数値化基準）

二層構造：地力係数層 ＋ 当日変動加算層 ／ v3

## 0. 全体の考え方（v3 の核）

各馬に「能力ベース点」を置き、**地力に効く要素は係数で掛け、当日その馬に起きた上下動は加算で足す**。最終点の序列で 軸／対抗／押さえ／消し に振り分ける。

v2 までは全要素を加算していたが、コース適性やバイアスは「地力に比例して効く」性質があるため係数化した。地力60の馬がコースにドンピシャでも勝ち負けには遠いが、地力90の馬がドンピシャなら別格になる——この非線形を加算では表現できないための改訂。

### 最終評価点の式

```
最終評価点 ＝ ［ ベース点 × コース適性係数 × バイアス係数 ］   ← 地力係数層
            ＋ 馬体重補正 ＋ ねじれ補正 ＋ 上がり3F補正 ＋ 枠補正   ← 当日変動加算層
```

- **係数層**：その馬の地力が、このコース・この馬場でどれだけ価値を持つか。地力に比例 → 掛け算。
- **加算層**：その日その馬に起きた上下動（体重・人気のねじれ・末脚の裏付け・枠の引き）。地力とは独立 → 加算。

### 合成係数のキャップ（重要）

コース適性係数 × バイアス係数の**合成値は 0.7〜1.5 の範囲にクリップする**。2つの係数が同方向に振れても、最大1.5倍・最小0.7倍を超えない。順位が一気に吹き飛ぶのを防ぐための安全弁。

```
合成係数 = clip( コース適性係数 × バイアス係数 , 0.7 , 1.5 )
最終点  = ベース点 × 合成係数 ＋（加算層の総和）
```

## 1. ベース点（主観評価）

ベース点はアシスタントの主観（過去走の内容・相手関係・展開適性・成長度などの総合判断）で 0〜100 点を付与する。前走着順やレーティングは参考に留め、機械的に決めない。目安は以下。

| 区分 | 点数帯 | イメージ |
|---|---|---|
| S（当該クラス断然） | 85〜100 | 能力上位かつ展開不問で崩れにくい |
| A（勝ち負け級） | 70〜84 | ハマれば勝ち切れる中心候補 |
| B（複勝圏あり） | 55〜69 | 展開・馬場が向けば2・3着まで |
| C（要素待ち） | 40〜54 | 条件が噛み合えば浮上、通常は押さえ |
| D（評価困難） | 0〜39 | 現級では足りない／状態不明 |

**注：** ベース点は「能力の核」の素点であり、当日要素（体重・オッズ・馬場）も、コース適性も、ここには織り込まない。コース適性は第4項の係数で、当日変動は第2・3・7項の加算で、それぞれ別途反映する。二重計上を避ける。上がり3F（末脚の質）も第7項で別途切り出すため、ベース点に重ねて織り込まない。

**v3 での注意：** ベース点には「このコース向きか」を含めない。純粋に地力・能力の核だけを測る。コース適性は係数側で評価するため、ベース点で適性を織り込むと二重に効いてしまう。

## 2. 馬体重増減の補正【加算層】

基準は ±0〜+8 を許容（補正なし）。さらに休み明け（中9週以上＝レース間隔が空いた）か、叩き2走目以降かで分岐する。プラス体重の意味が両者で逆になるため。

### 2-1. 通常時（叩き2走目以降 ／ 中8週以内）

| 増減（kg） | 補正 | 扱い |
|---|---|---|
| -4 〜 +8 | ±0 | 許容範囲。割引なし |
| +10 〜 +11 | -2 | 軽い注意 |
| +12 〜 +15 | -4 | 要確認。消しではない |
| +16 以上 | -8 | 大幅増。強い割引 |
| -6 〜 -9 | -3 | 急な絞り。状態を見極め |
| -10 以上 | -6 | 仕上げ過剰・消耗の懸念 |

### 2-2. 休み明け（中9週以上）

休み明けの増加は「成長・回復・余裕残し」の可能性があり、通常時より許容を広く取る。逆に大幅減は仕上がり途上のサインとしてやや重く見る。

| 増減（kg） | 補正 | 扱い |
|---|---|---|
| -2 〜 +12 | ±0 | 許容拡大。成長・回復分として容認 |
| +14 〜 +17 | -2 | 緩さの懸念。気配と併せて判断 |
| +18 以上 | -5 | 明らかな太め残り |
| -4 〜 -7 | -3 | 休み明けでの減は仕上げ途上を疑う |
| -8 以上 | -6 | 強い割引 |

**運用メモ：** 数値はあくまで初期割引。パドックの気配・毛艶・歩様で上下に微調整してよい。体重だけで軸を否定しない。

**気温・季節の扱い：** 暑熱期の入れ込み（気合過多）や大幅発汗による体重減は、この馬体重補正＋当日パドック気配で拾う。気温を独立の補正項にはしない。馬ごとの気温適性（夏に好走集中／寒さで動き渋い等）は固定特性として注目馬メモに記録し、その馬を評価するときだけ参照する（一律係数化しない＝「夏に強い血統」等の俗説はサンプル小・メンバー構成と交絡のため断定しない）。気温が馬場を高速化させる効果は隊列シミュレーション第1-2項（季節補正）で前残り指標として扱う。

## 3. オッズのねじれ補正（倍率比）【加算層】

単勝オッズ ÷ 複勝オッズ（上限側）の比率 R で判定する。R が大きいほど「単勝の割に複勝が買われている＝複勝圏の信頼が厚い」とみなし、複勝圏評価を加点する。R が小さい馬は単勝人気先行で、頭固定のリスクサインとして扱う。

**R ＝ 単勝オッズ ÷ 複勝オッズ（複勝は上限値を使用）**

| 比率 R | 補正 | 解釈 |
|---|---|---|
| R ≧ 4.0 | +4 | 強い妙味。複勝圏の信頼厚く押さえ〜対抗候補 |
| 3.0 ≦ R < 4.0 | +2 | 妙味あり。複勝で買われている |
| 2.0 ≦ R < 3.0 | ±0 | 標準的な関係。補正なし |
| 1.5 ≦ R < 2.0 | -2 | やや単勝人気先行。頭固定は慎重に |
| R < 1.5 | -3 | 単勝人気が突出。連下評価・頭固定回避サイン |

**補足：** 複勝オッズは幅で表示されるため上限値（高い方）で統一して計算する。締切直前のオッズで再計算するのが望ましい。Rの閾値は運用しながら調整する前提の初期値。R<1.5 は軸（頭固定）の回避サインであって、ヒモ（複勝圏）からの除外ではない。

**例：** 単勝12.0倍／複勝3.0倍 → R=4.0（+4）。単勝2.0倍／複勝1.4倍 → R≈1.43（-3、頭固定回避）。

## 4. コース適性係数【係数層・v3 新設】

v2 の「脚質×枠の相性補正」を発展させ、コースが要求する適性プロファイル全体を係数で評価する。脚質はこのプロファイルの一要素として吸収する（枠は第6項の加算へ分離）。

### 4-1. 考え方

各コースに「要求プロファイル」（そのコースで勝ち負けするのに必要な適性セット）を定義する。馬がそのセットをどれだけ満たすかで係数を決める。**地力（ベース点）に掛ける**ため、適性が噛み合うほど地力上位馬が跳ね、不向きなら強い馬でも地力を割り引かれる。

### 4-2. 係数表

| 充足度 | 係数 | 意味 |
|---|---|---|
| 完全充足（要求要素を全て満たす） | ×1.2 | 地力がフルに活きる。コースの申し子 |
| 主要素のみ充足 | ×1.1 | 概ね向く。大きな減点要素なし |
| 標準・判断保留 | ×1.0 | 素点どおり。可もなく不可もなく |
| 明確にミスマッチ（主要素を欠く） | ×0.85 | 強くても活き切らない。切れ味型が小回り等 |

### 4-3. 要求プロファイルは別ファイル参照

各コースの要求プロファイルと、脚質・適性ごとの充足判定は『コース別_脚質枠_補正表.md（v3）』に定義する。主要6コース（中山2000／東京2400／阪神2200内／京都2200外／東京1800／福島1800）は専用プロファイルを使用。それ以外のコースは下の汎用脚質プロファイルで代用する。

### 4-4. 汎用脚質プロファイル（専用コース以外）

専用プロファイル未定義のコースでは、脚質と「コースの主性質（瞬発型／持続型）」の噛み合わせで暫定係数を置く。

| | 瞬発型コース（長い直線・平坦） | 持続型コース（小回り・坂・短い直線） |
|---|---|---|
| 逃げ・先行 | ×1.0 | ×1.2（完全充足） |
| 差し（持続） | ×1.1 | ×1.1 |
| 差し（瞬発のみ） | ×1.2 | ×0.85（ミスマッチ） |
| 追込 | ×1.1 | ×0.85 |

## 5. 当日バイアス係数【係数層】

馬場最重視の方針に従い、当日の実馬場バイアスを地力係数層に掛ける。v2 の「脚質×枠補正に掛ける」方式から、「地力評価（ベース点×コース適性）全体に掛ける」方式へ変更。馬場が確定するのは発走直前のため、本線は固定せず複数シナリオを並行で持つ。

### 5-1. 係数表（符号反転は廃止）

v2 の外差し馬場 ×(-1.0)（符号反転）は二層構造では地力がマイナスになり破綻するため廃止。**外差し馬場は「コース要求プロファイルそのものが差し向きに変わる」と捉え、コース適性係数側（第4項）で評価を入れ替える**（前残り型を×0.85、持続差し型を×1.2 に）。バイアス係数は素直な増幅・減衰のみを担う。

| 当日バイアス | 係数 | 効果 |
|---|---|---|
| 強い内・前有利 | ×1.3 | 地の利を最大評価。前・内の地力上位馬を増幅 |
| 標準（内有利・前残り） | ×1.0 | 基準 |
| フラット（どの脚質も決まる） | ×1.0 | コース適性係数の差をそのまま反映 |
| 外差し・前崩れ馬場 | ×1.0（※第4項で前後入替） | バイアス係数は中立に置き、コース適性係数側で差し型を引き上げ・前残り型を引き下げる |

**運用：** 「内が止まり外が伸びる」当日実馬場を確認したら、バイアス係数を動かすのではなく、第4項コース適性係数の充足判定を差し型有利に組み替える。これにより合成係数が地力をマイナスにすることなく、馬場急変を表現できる。

### 5-2. 確認の手順

前レースで「内が止まり外が伸びているか」「逃げ・先行が残れているか」を必ず実馬場で確認し、係数（および第4項の充足判定）を最終決定する。当日の天候急変（雨・馬場悪化）で傾向が反転しうる点を常に念頭に置く。雨だけで前崩れと即断せず、ハイペース＋内荒れが揃って初めて前崩れトリガーとする。

## 6. 枠補正【加算層・v3 で分離】

v2 でコース適性に内包していた枠の有利不利を、加算層に小さく分離した。枠は地力とは独立の「引き」の要素であり、係数で増幅すると地力上位馬の枠の良し悪しを過大評価するため、フラットな加算（±2程度）に留める。

| 枠 | 標準 | 小回り・開幕週（内有利強） | 外枠不利コース（東京1800等） |
|---|---|---|---|
| 内（1〜2枠） | +1 | +2 | +2 |
| 中内（3〜4枠） | ±0 | +1 | +1 |
| 中外（5〜6枠） | ±0 | ±0 | -1 |
| 外（7枠〜） | -1 | -2 | -2 |

**注：** コース固有の枠癖（福島4枠不振、東京1800の外枠包まれなど）は各コースのプロファイル補足に従い微調整。枠補正は加算層なので、地力の高低にかかわらず一律に効く。

## 7. 上がり3F（末脚）の評価【加算層】

過去走の上がり3F（ラスト600m）を「末脚の質」の指標として加算補正する。本体は相対評価（同レース内順位／レース上がり比）で馬場・ペースのノイズを排除し、絶対値は限定的なボーナス／据え置き条件として併用する。上限は ±0〜+3。能力本体はベース点で測り、本項は末脚の裏付け／減点に限定する（二重計上回避）。

### 7-1. 評価に使う指標（優先順）

1. 上がり最速・1〜3位の回数（直近5走）＝ そのメンバーで最も速い脚を使えたか
2. レース上がり（勝ち時計の上がり）との差 ＝ -0.3秒以上速ければ優秀
3. 着順と上がりの乖離 ＝「負けたが上がり最速」は展開不利の好走、次走加点材料

### 7-2. 補正の目安（直近3〜5走ベース）

| 末脚の評価 | 補正 | イメージ |
|---|---|---|
| 重賞・上位人気戦で上がり最速を複数回 | +3 | 末脚の裏付け強。差し・追込の軸候補 |
| 上がり上位（1〜3位）が常連 | +2 | 安定した決め手 |
| 平均的（メンバー中位の上がり） | ±0 | 補正なし |
| 上がりが遅い／詰まる脚質だが先行で結果 | ±0 | 先行・粘り型は減点しない（後述） |
| 速い脚を全く使えていない差し・追込馬 | -2 | 脚質と末脚が不一致。展開待ちで割引 |

### 7-3. 絶対値の併用ルール（相対評価への上乗せ／据え置き）

相対評価を本体としつつ、絶対上がりタイムで以下を加味する。生タイムを直接足さず、あくまで相対評価の補強・抑制に使う。

| 条件 | 扱い |
|---|---|
| 時計のかかる条件（重馬場・ハイペース・非高速馬場）で明確に速い絶対上がり（芝で33秒台前半等）を出した実績 | 相対評価に +1 ボーナス（合計上限+3は維持） |
| 低レベル少頭数スロー戦でのみ上がり最速 | 相対順位が高くても加点据え置き（+0）。誰でも速くなるため価値を割引 |

**狙い：** 相対評価のノイズ耐性を保ちつつ、「本物の決め手＝絶対値も速い馬」を拾い直し、「低レベル戦の見かけ上の最速」を過大評価しない。

### 7-4. 脚質・コースとの連動（重要）

- **先行・逃げ馬は上がりが遅くて当然** → 上がりの遅さで減点しない。前で粘る馬は「上がりタイム」でなく「失速の小ささ」で見る。減点対象は「差し・追込なのに速い上がりを使えていない馬」に限る。
- **高速上がりコース（東京・京都／毎日王冠・JC・ダービー等）** → 上がり補正の効き目をそのまま（+3まで）。決め手が直結する。
- **持続力コース（中山・阪神内回り・福島／皐月賞・宝塚等）** → 瞬発的な上がり最速より「長く良い脚」を重視。上がり補正は最大+2に圧縮し、ロングスパート適性を優先。
- **コース適性係数と連動：** 持続型コースで「瞬発のみの差し馬」は第4項で×0.85を受けるため、上がり加算と係数割引が両方効いて自然に序列が下がる。二重ペナルティに見えるが、係数は地力の活き方、加算は末脚の質と別軸を測っているため意図どおり。

### 7-5. 運用上の注意

- 上がりタイムは公式データ・netkeiba等で確認。相対評価が本体、絶対値は補強。
- 「上がり最速だが毎回4〜5着」＝ 詰めの甘さ。頭固定回避サイン（R補正と併せて判断）。
- 数値は初期値。結果を記録しながら補正幅を調整する。

## 8. 最終点 → 振り分けの目安

| 区分 | 位置づけ | 扱い |
|---|---|---|
| 軸 | 最上位1〜2頭 | 三連系の軸。ただし頭固定はR・展開リスクで再検討 |
| 対抗 | 次点2〜3頭 | 軸と入れ替わりうる勝ち負け級 |
| 押さえ | 複勝圏候補 | 2・3着付け。妙味（R高）馬を優先的に拾う |
| 消し | 下位 | 理由を必ず明示（能力・展開・体重・馬場不適など） |

**頭固定の判断：** 能力上位でも R<1.5（単勝人気突出）や展開リスクがある場合は頭固定を外し、軸を勝ち切らない前提の買い目（複系の流し）も併走させる。

## 9. 計算例（v3 の流れ）

例：阪神内2200、地力A評価の持続差し馬。ベース点78。コース要求は「持続力・坂適性・上がり上位」。

- ベース点：78
- コース適性：持続差しで主要素充足 → ×1.1
- 当日バイアス：標準 → ×1.0
- 合成係数：1.1×1.0 = 1.10（キャップ内）
- 地力係数層：78 × 1.10 = 85.8
- 加算層：馬体重±0、R=3.5（+2）、上がり3F最速複数だが持続型コースで+2圧縮、内枠+1 → 加算 +5
- 最終点：85.8 + 5 = 90.8

同コースで地力S（ベース90）の「瞬発のみ差し馬」：
- 90 × 0.85（ミスマッチ）× 1.0 = 76.5、加算+3 → 79.5
- 地力では上だが、コース不適の係数割引で逆転される。これが v3 の狙い。

## 10. 妙味スコア（穴馬救済トラック・v3 新設）

### 10-1. なぜ別トラックが要るか

第1〜9項の地力評価（ベース点×係数＋加算）は、人気馬・実力馬を正確に測る一方、2桁人気の穴馬を構造的に取りこぼす。穴馬はベース点（地力）が低く付くため、係数1.2を掛けても上位に届かない。掛け算は元が小さいと伸びしろも小さいからだ。

だが穴の好走は「地力が高いのに人気がない」からではなく、「特定条件が噛み合うと地力以上に走る」から起きる。これは地力の物差しでは測れない。よって**地力評価とは独立した第二の物差し＝妙味スコアを並走させ、☆だけはこのスコアで拾う**。

### 10-2. 共通トリガー表（全コース固定）

穴の普遍的な原理。コースに依存せず、一度作って全レース共通で使う。各トリガーに基礎点を置く。

| 共通トリガー | 基礎点 | 趣旨 |
|---|---|---|
| 人気薄（6番人気〜）の前向き脚質（逃げ・先行） | 2 | 人気薄の前残りは回収率が跳ねる |
| R値ねじれ大（R≧4.0） | 2 | 単勝の割に複勝が買われている＝密かな信頼 |
| 軽ハンデ（53kg以下目安）の上昇馬 | 2 | ハンデ戦の構造的有利。軽量で前に行ける |
| 馬場・展開が向く枠の引き（内枠×前残り型 等） | 2 | 地の利を引いた人気薄 |
| 巧者騎手・有効な前走ステップ | 1 | コース実績騎手／買えるローテ |

### 10-3. コース係数（レース毎に可変・別ファイル参照）

各トリガーがそのコースでどれだけ効くかをコース係数で調整する。係数は『コース別_脚質枠_補正表.md』の各プロファイルに「妙味係数」として1行で定義する（前残り型か差し型かで前残りトリガー／末脚トリガーの係数が反転）。

```
妙味スコア = Σ（共通トリガー基礎点 × そのコースの係数）
```

- 前残り型コース（福島1800・中山2000等）：前向き脚質トリガー×1.5 / 末脚トリガー×0.5
- 差し型コース（東京2400・東京1800等）：前向き脚質トリガー×0.5 / 末脚トリガー×1.5

### 10-4. 妙味スコアの使い方

| 妙味スコア | 扱い |
|---|---|
| ≧ 5 | 地力で下位でも ☆ として強制的に拾う。コース適性係数の下振れ（×0.85）を免除し×1.0で止める |
| 3〜4 | ヒモ（3着付け）候補として保持 |
| < 3 | 通常どおり地力評価に委ねる |

### 10-5. 買い目での安全網

評価でこぼしても買い目で拾う。前残り型・荒れやすいレース（福島1800のラジオNIKKEI賞等、毎年7番人気以下が3着内に激走）では、**穴・強気型の買い目に妙味スコア上位の人気薄を1〜2頭、3着付けで機械的に確保する**。堅実型と穴・強気型を分ける既存方針の、穴側の芯にする。

### 10-6. 運用上の注意

- 妙味スコアは地力評価を**置き換えない**。☆の選定にのみ使い、◎○▲△は従来どおり地力評価の序列で振る。
- 穴は減点で消さない（不安材料は割引材料、軸ごと否定しないの既存原則の延長）。妙味スコア閾値超えの馬に係数下振れを適用しないのはこのため。
- 基礎点・係数・閾値は初期値。結果を記録しながら調整する。

### 10-7. ペーススコア連動の前後反転（v3 バックテスト由来の補強）

**背景：** 福島1800等を「前残り型」と固定プロファイルすると、ハイペースで前が崩れた年に妙味トラックが機能しない。実例：2025年（平均ペース・前残り）はインパクトシー（9番人気3着）を前残り型妙味で的中できたが、2024年（ハイペース・前崩れ）は最後方差しのオフトレイル（6番人気）が勝ち、前残り型妙味では勝ち馬を拾えず、前向き脚質の人気薄（ミナデオロ・ジュンゴールド）が8〜10着に飛んで無駄打ちになった。

**対策：** 妙味係数（前残りトリガー／末脚トリガーの重み）を固定せず、**ペーススコアで前後を反転させる**。

#### ペーススコアの算出（枠順確定時点・機械的）

出走各馬の脚質を分類し、以下で算出する。

```
ペーススコア = （明確な逃げ馬数 × 2）＋（先行馬数 × 1）
```

| ペーススコア | ペース判定 | 妙味係数の扱い |
|---|---|---|
| ≦ 3 | 前残り濃厚（同型少・単騎or不在） | 前残り型のまま（前向き×1.5 / 末脚×0.5）。前残り妙味を強化 |
| 4〜6 | 中立 | コース標準のプロファイルどおり |
| ≧ 7 | 前崩れ警戒（同型多数・ハナ争い濃厚） | **前後反転**（前向き×0.5 / 末脚×1.5）。差し型の人気薄を妙味で拾う |

#### 開催日による補正

- 開幕週（ラジオNIKKEI賞は毎年開幕2日目）＝構造的に内・前有利スタート。ペーススコアが中立（4〜6）なら前残り型に寄せてよい。
- 開催後半・荒れ馬場＝差し台頭。中立でも末脚型へ寄せる。

#### 当日実馬場による最終確定（人間判断・2割）

ペーススコアと開催日で事前フラグ（前残り/中立/前崩れ）を立てたうえで、**発走前に前レースの実馬場（内が止まり外が伸びているか／前が残れているか）を目視確認し、フラグを確定または反転する**。馬場急変（雨・内荒れ）はペーススコアでは読めないため、ここは原理的に当日確認が必須。雨単独では反転せず、ハイペース（ペーススコア≧7）＋内荒れが揃って初めて前崩れを確定する。

#### 二段構えのまとめ

```
【金曜・枠順確定】機械判定
  ペーススコア算出 ＋ 開催日 → 事前フラグ（前残り/中立/前崩れ）
  → 妙味係数の前後を仮設定

【日曜・発走前】実馬場確認
  前レースで内が止まっていないか目視
  → 事前フラグを確定 or 反転 → 妙味係数を最終確定
```

この反転は第5項（当日バイアスの前後入替）・最後の4F隊列シミュレーション（ペーススコアを入口に算出）と連動させる。

## 11. レース荒れ度ランク（v3 新設）

### 11-1. 趣旨と最重要原則

過去10年の配当分布から、各重賞に「荒れやすさ」のランクを振る。荒れやすいレースで穴を取りにいくための指標だが、**使い方を誤ると精度を落とす**ため、最重要原則を先に置く。

**【絶対原則】荒れ度はベース点（地力）に絶対に触れさせない。** 「荒れるレースだから人気薄を一律加点」は誤り。荒れ度統計は「平均して荒れる」を示すだけで、「どの人気薄が来るか」は何も教えない。人気薄全体を持ち上げると、来ない多数を一緒に上げて評価が薄まり、無駄打ちが増えて回収率はむしろ悪化する（2024年バックテストで実証：前残り妙味の一律拾いで来たのは1頭、他は8〜10着）。

穴は「全体を持ち上げて」ではなく「妙味トラック（第10項）で1〜2頭に絞って」拾う。荒れ度はその絞り込みを後押しする補助情報に徹する。

### 11-2. 荒れ度ランクの定義

過去10年の「3連単の最低・平均配当」「1番人気の3着内率」「6番人気以下の馬券絡み数」から3段階で振る。各レースのランクは『レース別_過去傾向データ.md』に明記する。

| ランク | 目安 | 例 |
|---|---|---|
| 高（荒れる） | 3連単が常に万馬券／1番人気3着内率が3割前後／毎年人気薄が絡む | ラジオNIKKEI賞、エリザベス女王杯、宝塚記念 |
| 中 | 上位人気中心だが中穴も突っ込む | 皐月賞、府中牝馬S |
| 低（堅い） | 1〜5番人気で決まりやすい | 日本ダービー、ジャパンC |

### 11-3. 荒れ度の効かせ方（3点のみ・ベース点以外）

荒れ度ランクは以下3つの運用にのみ接続する。いずれも個別馬のベース点をいじらない。

**① 人気馬の信頼度を下げる（頭固定回避の強化）**
荒れ統計が本当に示すのは「1番人気・上位人気が飛びやすい」。荒れ度「高」のレースでは、第8項の頭固定判断をより厳しくし、1番人気の頭固定を原則外す。これは人気薄を上げるのではなく人気馬を割り引く方向。最も確実な使い方。

**② 買い目の予算配分を穴側へ寄せる**
荒れ度「高」では、堅実型の比重を下げ穴・強気型へ予算を多めに振る。個別評価は一切いじらず資金配分だけ動かすため、評価精度を落とさず期待値を取れる。

**③ 妙味スコアの採用閾値を1段下げる（限定的）**
荒れ度「高」のレースに限り、第10-4項の☆採用閾値を5→4に下げ、☆枠を1頭広げる。人気薄全体の底上げではなく、妙味トラックで既に絞った中の採用幅を1つ広げるだけなので、無駄打ち増を最小限に抑えられる。

| 荒れ度 | ①人気馬割引 | ②予算配分 | ③妙味閾値 |
|---|---|---|---|
| 高 | 1番人気の頭固定を原則外す | 穴・強気型を厚く | 5→4に緩和 |
| 中 | 標準（R<1.5等で個別判断） | 堅実：穴＝標準 | 5のまま |
| 低 | 上位人気を素直に軸 | 堅実型を厚く | 5のまま |

### 11-4. 運用上の注意

- 荒れ度は「レースの型」の属性であり、当年メンバーで上下しうる（抜けた1強がいれば荒れ度「高」でも堅くなる）。ランクは初期値、当年の構成で微調整。
- 荒れ度「高」でも、1番人気を「消す」のではなく「頭から外す（ヒモには残す）」。極端な人気軽視はしない。
- ベース点不可侵の原則を常に最優先する。

## 運用上の注意

- 数値・係数はすべて初期値であり、結果を記録しながら閾値・補正幅・係数レンジを調整していく前提。
- 体重・オッズ・馬場・上がり3Fの単一要素で軸を全否定しない。割引材料として扱う。
- 係数は地力（ベース点）にのみ掛け、当日変動は加算に留める。この二層を混ぜない。
- 合成係数は必ず 0.7〜1.5 にクリップしてから掛ける。
- 馬券は自己責任。予算は無理のない範囲で。
