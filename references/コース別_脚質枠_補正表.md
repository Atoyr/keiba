# コース別 要求プロファイル＆コース適性係数表

主要重賞コース（6コース）／ 評価ルール第4項（コース適性係数）の定義版 ／ v3

**使い方：** 各コースに「要求プロファイル」（勝ち負けに必要な適性セット）を定義する。馬がそのセットをどれだけ満たすかで、コース適性係数（×1.2／×1.1／×1.0／×0.85）を決め、ベース点に掛ける。枠の有利不利は係数ではなく評価ルール第6項の枠補正（加算層）で扱う。当日バイアス係数（第5項）を合わせて掛け、合成係数は 0.7〜1.5 にクリップする。

**v2 からの変更：** 「脚質×枠の加算表」を廃止し、「要求プロファイル → 充足度 → 係数」方式へ移行。脚質はプロファイルの一要素として吸収。枠は加算層へ分離。

---

## 充足度判定の共通ルール

各コースのプロファイルは「主要素」と「加点要素」で構成する。

- **完全充足（×1.2）**：主要素を全て満たし、かつ加点要素も備える
- **主要素充足（×1.1）**：主要素を満たすが加点要素は中立
- **標準（×1.0）**：主要素の一部のみ、判断保留
- **ミスマッチ（×0.85）**：主要素を明確に欠く（例：持続型コースで瞬発一辺倒）

## ① 中山 芝2000m

**主な重賞：** 皐月賞、中山記念、オールカマー、弥生賞、ホープフルS、中山金杯

**形態：** 内回り・右回り・小回り。1コーナーまで約405m、直線約310mと短い、ゴール前に高低差2.4mの急坂、これを2度上るタフな構成。高低差5.3mはJRA最大。

**コース性質：** 持続型（小回り＋短い直線＋急坂）

**要求プロファイル：**
- 主要素：先行力（4角4番手以内を取れる）／ 急坂をこなすパワー ／ コーナーでの器用さ
- 加点要素：ロングスパート持続力（早めに動いて粘れる）／ 渋った馬場への対応
- ミスマッチ：瞬発力一辺倒の差し・追込（短い直線で届かない）

| 適性タイプ | 係数 |
|---|---|
| 先行・好位＋パワー＋器用さ完備 | ×1.2 |
| 先行〜中団の持続型 | ×1.1 |
| 標準・位置取り保留 | ×1.0 |
| 瞬発のみの差し／後方一辺倒の追込 | ×0.85 |

**補足：** マクリ巧者（戸崎など）が中団にいる場合は持続差しとして×1.1まで戻してよい。秋9月開催は野芝で高速化し前残り強化＝先行型を×1.2寄りに。枠は加算層で内+1〜2（小回り内有利）。

**妙味係数（前残り型）：** 前向き脚質トリガー×1.5 / 末脚トリガー×0.5 / R・軽ハンデ・枠の引きトリガー×1.0。小回り・短い直線・急坂で人気薄の前残りが妙味。

## ② 東京 芝2400m

**主な重賞：** 日本ダービー、ジャパンC、オークス、青葉賞

**形態：** 外回り・左回り・広い。1コーナーまで約350m、直線525.9m（国内2位）、直線に高低差2.1mの長い坂。コーナー緩やか。

**コース性質：** 瞬発型（長い直線・末脚勝負）

**要求プロファイル：**
- 主要素：末脚の確実性（上がり上位を恒常的に使える）／ 長い直線で伸びる瞬発力 ／ 2400mをこなす底力・スタミナ
- 加点要素：インをロスなく立ち回るポジション勘 ／ 坂で甘くならない持続力
- ミスマッチ：先行・粘り一辺倒で末脚を欠く（長い直線で差される）

| 適性タイプ | 係数 |
|---|---|
| 確実な末脚＋スタミナ＋立ち回り完備 | ×1.2 |
| 末脚上位の差し・好位差し | ×1.1 |
| 標準・展開待ち | ×1.0 |
| 逃げ・先行で末脚を全く欠く | ×0.85 |

**補足：** 枠より「インをロスなく立ち回れるか」を重視（加算層で内+1）。スロー想定なら前目有利に振れるため先行型を×1.0据え置き、ハイペース・多頭数なら差し優位＝差し型×1.2寄り。馬場の影響を受けやすく当日バイアス確認が特に効く。

**妙味係数（差し型）：** 末脚トリガー×1.5 / 前向き脚質トリガー×0.5 / R・軽ハンデ・枠の引きトリガー×1.0。長い直線で人気薄の末脚が突っ込む形が妙味。前残りの穴は信頼薄。

## ③ 阪神 芝2200m（内回り）

**主な重賞：** 宝塚記念、エリザベス女王杯（阪神開催時）

**形態：** 内回り・右回り。外回り4コーナー出口からスタート、1コーナーまで約525mと長い。スタート直後が下り坂で前半速くなりやすい。直線約357mと平均的、残り200m付近に高低差1.8mの急坂。坂を2度上る。

**コース性質：** 持続型（内回り＋短め直線＋坂2回）。ただし下り坂スタートで前傾化すると差しも届く二面性。

**要求プロファイル：**
- 主要素：先行〜好位の持続力 ／ 急坂をこなすパワー ／ 長く良い脚（ロングスパート）
- 加点要素：渋った馬場・ハイペース耐性（宝塚は梅雨で渋りやすい）／ 上がり上位（前傾化した年に効く）
- ミスマッチ：瞬発一発の追込（短い直線で決まりにくい）

| 適性タイプ | 係数 |
|---|---|
| 先行・好位＋パワー＋持続力完備 | ×1.2 |
| 持続型の差し・好位差し | ×1.1 |
| 標準・展開保留 | ×1.0 |
| 瞬発のみの追込 | ×0.85 |

**補足：** 馬場が渋る・ハイペース濃厚なら持続力型差し馬を×1.2まで引き上げ（二面性の発露）。この組み替えが v3 における「前崩れ馬場」の表現＝バイアス係数ではなくプロファイル側で前後を入れ替える。秋開催は野芝高速化で前・内をさらに優遇。枠は加算層で内+1。

**妙味係数（前残り寄り・二面性）：** 標準時は前向き脚質トリガー×1.2 / 末脚トリガー×0.8。ただし馬場渋り・ハイペース濃厚（宝塚の梅雨など）の年は前後反転し前向き×0.8 / 末脚×1.4（持続差しの人気薄が突っ込む）。R・軽ハンデ・枠トリガー×1.0。

## ④ 京都 芝2200m（外回り）

**主な重賞：** エリザベス女王杯（京都開催時）、京都記念、京都新聞杯

**形態：** 外回り・右回り。1コーナーまで約397mと長く隊列が整いやすい。3コーナーに高低差約4.3mの「淀の坂」、下りで加速。直線約400mで坂はなく平坦。高速決着になりやすい。

**コース性質：** 持続＋瞬発の複合（淀の坂下りで自然加速、平坦直線）

**要求プロファイル：**
- 主要素：中団以前の位置取り（後方一気は届きにくい）／ 下りで動ける器用さ・持続力 ／ 長く良い脚
- 加点要素：高速決着への対応（速い時計）／ 平坦でモタつかない反応
- ミスマッチ：極端な後方脚質（位置取りが取れない追込）

| 適性タイプ | 係数 |
|---|---|
| 好位〜中団＋下りで動ける持続力完備 | ×1.2 |
| 先行・中団の立ち回り型 | ×1.1 |
| 標準・位置保留 | ×1.0 |
| 極端な後方一気の追込 | ×0.85 |

**補足：** 枠より位置取り（中団以前）を重視。下りで動ける器用さ・持続力を加点。極端な後方脚質は割引のまま。枠は加算層で標準扱い（枠差小）。

**妙味係数（複合・やや差し）：** 末脚トリガー×1.2 / 前向き脚質トリガー×0.8。淀の坂下りで中団から動ける人気薄が妙味。ただし後方一気は届きにくいので追込の穴は割引。R・軽ハンデ・枠トリガー×1.0。

## ⑤ 東京 芝1800m

**主な重賞：** 府中牝馬S、毎日王冠、エプソムC、共同通信杯、東スポ杯2歳S

**形態：** 左回り。1〜2コーナー奥のポケットからスタートし、斜めに150mほど進んで向正面へ入る特殊な形。スタート直後にコーナーがあるため外枠が不利。直線525.9mと長く、高低差2.1mの坂。3コーナーまで約750mと長くワンターン。

**コース性質：** 瞬発型（長い直線・末脚勝負）。ただし外枠不利が明確。

**要求プロファイル：**
- 主要素：末脚の確実性（長い直線で伸びる）／ スタート〜序盤のポジション取り（外枠だと包まれる）
- 加点要素：坂で甘くならない持続力 ／ 前半スロー〜瞬発戦への対応
- ミスマッチ：外枠を引いた差し・追込（位置取りで届かない）／ 末脚を欠く逃げ・先行

| 適性タイプ | 係数 |
|---|---|
| 確実な末脚＋好位取り＋（内目）完備 | ×1.2 |
| 末脚上位の差し・好位差し | ×1.1 |

**妙味係数（差し型・牝馬戦は補正）：** 末脚トリガー×1.4 / 前向き脚質トリガー×0.6。長い直線で人気薄の末脚が妙味。ただし府中牝馬など牝馬ハンデ前哨戦では牝馬の前残りが起きやすく、内枠を引いた軽ハンデ先行馬は前向き脚質トリガー×1.0まで戻す。R・軽ハンデ・枠トリガー×1.0。

## ⑥ 福島 芝1800m（v3 新規）

**主な重賞：** ラジオNIKKEI賞（3歳ハンデG3・開幕週）、福島記念

**形態：** 右回り・小回り。スタンド前スタートで1コーナーまで約300m＝序盤からポジション争いが起きやすい。小回り4コーナー、外を回されるとロス大。直線292mと短い。

**コース性質：** 持続型（小回り＋短い直線）。瞬発力勝負では届きにくく、機動力・持続力勝負。府中1800（瞬発型）とは別物。

**要求プロファイル：**
- 主要素：先行力（4角4番手以内）／ 小回りの機動力・器用さ ／ 早めに動いて粘る持続力
- 加点要素：内枠を引いてロスなく立ち回れる ／ 軽ハンデ（3歳ハンデ戦＝軽量上昇馬有利）／ 福島巧者騎手（田辺・津村・荻野極ら関東騎手）
- ミスマッチ：瞬発一辺倒の差し・後方一気の追込（短い直線・小回りで届かない）

| 適性タイプ | 係数 |
|---|---|
| 先行・好位＋機動力＋持続力完備（軽ハンデなら尚良） | ×1.2 |
| 先行〜中団の立ち回り型 | ×1.1 |
| 標準・位置保留 | ×1.0 |
| 瞬発のみの差し／後方一気の追込 | ×0.85 |

**補足（傾向データ・過去10年）：**
- 脚質：先行【4-7-0-23】が中心、逃げ【1-2-3-4】は複勝率60%だが頭数少なく妙味（穴）寄り。差し・追込は展開待ち。勝ち馬は全て4角4番手以内。
- **逃げ馬の人気別分岐：** 人気薄の逃げは☆妙味（単複回収率200%超）、人気（〜5番人気）の逃げは◎堅実軸へ昇格。ただしハンデ重＋単騎可否を最後の4F隊列シミュレーションで要確認。
- 枠：開幕週小回りで内有利が強い。1〜2枠は加算層で+2、3枠+1。**4枠のみ過去10年3着内なし＝加算層で4枠は別途-1の癖補正**。
- 人気：1番人気3着内率30%と不振。10番人気以下は2着3着が稀にあるのみ。既存の3歳・牝馬ハンデ戦ルール（軽ハンデ上昇馬有利・favorites less reliable）がそのまま適用可能。
- 前走ステップ：1勝クラス・リステッド勝ちの上がり馬を重視。前走重賞好走馬の勝利は過去10年で1回のみ。ただし重賞連対経験なしの1勝馬は割引（通算2勝以上が望ましい）。
- 血統：サンデーサイレンス系が中心（8勝）。ハーツクライ系は不振傾向。

**妙味係数（前残り型・荒れやすい）：** 前向き脚質トリガー×1.5 / 末脚トリガー×0.5 / R・軽ハンデ・枠の引きトリガー×1.0。
- このレースは穴の安全網を特に厚く運用する：毎年7番人気以下が3着内に激走し、3連単は最低でも2万円超。妙味スコア上位の人気薄を穴・強気型の3着付けに機械的に1〜2頭確保。
- 人気薄の逃げ・先行（単複回収率200%超の源泉）は妙味スコアが乗りやすい。内枠（1〜2枠）×先行は枠の引きトリガーも重なり高スコアになりやすい。
- 軽ハンデ53kg以下の上昇馬（前走1勝/L勝ち）は軽ハンデトリガー＋ステップトリガーで拾える。3歳ハンデ戦の構造的有利と一致。

## 共通の運用メモ

- コース適性係数は標準良馬場の要求プロファイルを基準にする。当日バイアス係数（強い内前×1.3／標準×1.0／外差し・前崩れは第4項プロファイル側で前後入替）を合わせて掛け、合成係数を 0.7〜1.5 にクリップ。
- 外差し・前崩れ馬場は、バイアス係数を動かすのではなくコース適性プロファイルの充足判定を差し型有利に組み替えて表現する（符号反転の廃止に伴う運用）。
- 開催時期で芝が変わる（秋の野芝は高速＝前・内強化、開催後半の荒れ馬場はタフ＝差し台頭）。時期を加味して微調整。
- 枠の有利不利は係数ではなく評価ルール第6項の枠補正（加算層）で扱う。コース固有の枠癖（福島4枠不振、東京1800外枠不利）はここの補足を反映。
- 数値・係数は初期値。結果を記録しながらコースごとに調整していく。
- 馬券は自己責任。予算は無理のない範囲で。
